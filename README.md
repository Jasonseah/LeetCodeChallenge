# LeetCodeChallenge
Daily Leet Code challenge for leaning algorithm and land a job in Google 


## To build and run
> to build
```
docker build -t my-golang-app .
```

> to run 
```
docker run -it --rm --name leet-code-challenge my-golang-app
```
